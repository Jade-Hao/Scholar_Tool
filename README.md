# Scholarship Tool

A simple, student-focused platform designed to make finding and organizing outside scholarships easier, faster, and less stressful.
---
# Overview

Searching for scholarships can be overwhelming for both students and parents. Scholarship Tool aims to simplify that process by providing a clean, organized interface where users can track scholarship opportunities in one place.

The ***current goal*** is to build a lightweight MVP inspired by a spreadsheet-style workflow — similar to Google Sheets — while adding smarter filtering and organization features tailored specifically for scholarship searches.
---
# Planned Features
- Spreadsheet-style scholarship tracker
- Add and organize scholarship opportunities
### Search and filter scholarships by:
- Due date
- Merit-based or need-based
- Major-specific scholarships
- Eligibility requirements
- Award amount
- Application status
- Easy-to-read dashboard layout
- Student and parent-friendly UI

***The first version of the project focuses on creating a basic but functional scholarship management system where users can:***

View scholarships in a table layout
Add and edit scholarship entries
Filter scholarships using keywords and categories
Keep track of deadlines and application progress
Vision

***The long-term vision is to create a centralized scholarship discovery and organization platform that reduces stress during the college funding process and helps students avoid missing valuable opportunities.***
